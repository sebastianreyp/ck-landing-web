# Capital Kitchen Landing

Landing corporativa en Next.js + TypeScript + Tailwind para Capital Kitchen, pensada para despliegue en Vercel.

## Stack

- Next.js App Router
- React + TypeScript
- Tailwind CSS
- pnpm

## Ejecutar localmente

1. Instala Node.js.
2. Instala pnpm si no lo tienes.
3. Ejecuta:

```bash
pnpm install
pnpm dev
```

## Build de producción

```bash
pnpm build
pnpm start
```
